# Nexus Earth Game Design Document

## Overview

Nexus Earth is a next-generation open-world RPG where every player creates their own journey.

Players are not locked into one path.

They can become entrepreneurs, explorers, fighters, leaders, creators, or anything the world allows.

## Core Idea

A living world where player decisions affect their future.

## Player Possibilities

Players can:

- Own property
- Create businesses
- Drive vehicles
- Explore cities
- Complete missions
- Build reputation
- Join organizations

## World Design

The world should feel alive.

NPCs should have:

- Jobs
- Schedules
- Personalities
- Goals

## Development Philosophy

Build small working systems first.

Expand the world over time.

Every system should be modular and expandable.
