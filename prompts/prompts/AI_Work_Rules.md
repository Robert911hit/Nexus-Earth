# Nexus Earth AI Work Rules

## Main Rule

Every AI worker must read the Master Blueprint before creating anything.

## File Rules

Keep files organized.

Do not delete another worker's work.

Do not create random technology.

Follow the approved technology stack.

## Development Rules

Build small features first.

Test before adding more features.

Document important changes.

## GitHub Rules

All work must be saved in GitHub.

Use clear commit messages.

## Team Roles

Planning and architecture.

Story and world creation.

Coding AI:
Programming and debugging.

Lovable:
Website and web systems.
